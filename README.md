
# FakeStore React

A simple e-commerce frontend built with **React** and **Vite** that fetches product data from the Fake Store API and displays it in a clean, Amazon-inspired card layout.

## Features

* Fetches products dynamically from the Fake Store API.
* Displays products as responsive cards with images, titles, prices, and details.
* Category filter using a dropdown menu.
* Supports categories such as:

  * Men's Clothing
  * Women's Clothing
  * Jewelry
  * Electronics
* Fast development experience powered by Vite.

## Tech Stack

* React
* Vite
* JavaScript
* CSS
* Fake Store API

## Getting Started

```bash
npm install
npm run dev
```

The application will start locally and can be viewed in your browser.
